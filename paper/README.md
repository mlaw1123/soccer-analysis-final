# Paper build

`main.tex` is the arXiv-ready source and `main.pdf` is the verified rendered
paper. It cites result values only from `artifacts/final_run/`.

Build with Tectonic 0.16.9 or a conventional LaTeX/BibTeX toolchain:

```bash
cd paper
tectonic main.tex
```

The figures are vector PDFs generated by the experiment. After every substantive
paper change, render all PDF pages to images and inspect them for clipping,
overlap, unreadable text, and broken references before committing `main.pdf`.

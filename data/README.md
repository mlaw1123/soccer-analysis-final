# Data

`input/matches.parquet` is a normalized subset of the StatsBomb Open Data match
metadata inherited from the earlier `Soccer-Probability-Engine` work. It keeps
only identifiers, dates, competition/season/team names, and final scores needed
to reconstruct labels and strictly prematch features.

`input/dataset_manifest.json` records the upstream URL, source and normalized
SHA-256 hashes, row/date coverage, label counts, and schema.

`derived/` is generated by the feature pipeline. The current result is committed
because it is small and permits direct audit, but it must be reproducible from
the normalized input.

StatsBomb's repository terms require attribution and state that the data are
made available for research and genuine interest in football analytics. Consult
the upstream repository for the governing current terms.
